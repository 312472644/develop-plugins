## 描述

vite打包后生成压缩文件zip。

## 参数说明

| 参数名            | 说明               | 类型    | 默认值   |
| ----------------- | ------------------ | ------- | -------- |
| targetPath        | 构建生成目标文件夹 | string  | `'dist'` |
| filename          | 生成压缩包名称     | string  | `'dist'` |
| deleteSourceFiles | 是否删除源文件夹   | boolean | `true`   |
