## 描述

webpack打包生成压缩文件。

## 参数说明

| 参数名            | 说明               | 类型    | 默认值   |
| ----------------- | ------------------ | ------- | -------- |
| targetPath        | 构建生成目标文件夹 | string  | `'dist'` |
| filename          | 生成压缩包名称     | string  | `'dist'` |
| deleteSourceFiles | 是否删除源文件夹   | boolean | `true`   |

vite版本[点击这里](https://www.npmjs.com/package/@develop-plugins/vite-generate-zip)。
